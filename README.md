## Getting Started
### Installation
1. Clone the repo
```sh 
git clone https://github.com/alexandertolchinsky/erpaero.git
```
2. Prepare .env file
```sh
cp .env.example .env
```
3. Edit .env file
```sh
vim .env
```
4. Install package locally
```sh
make setup
```
## How to run

```sh 
make start
```
